## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Build Electron app
npm run electron:build:all
```

- React + Vite
- Redux Toolkit
- Firebase (Auth, Firestore, Hosting)
- Service Worker for offline support
